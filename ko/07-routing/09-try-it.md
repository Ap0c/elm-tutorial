# 실행해 보기

지금까지 한 걸 테스트해 봅시다. 어플리케이션을 실행합니다:

```bash
nf start
```

브라우저에서 `http://localhost:3000` 을 엽니다. 플레이어 목록을 볼 수 있습니다.

![screenshot](09-list.png)

`http://localhost:3000/#players/2` 로 가면 플레이어 하나를 볼 수 있습니다.

![screenshot](09-edit.png)
