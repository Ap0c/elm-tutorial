# 試してみよう

さあ試してみましょう！ ある端末でフェイクAPIサーバを実行するには：

```bash
npm run api
```

そして、(実行していない場合は)アプリケーションを実行します。

```bash
npm run dev
```

ブラウザをリフレッシュすると、アプリケーションはサーバーからプレーヤーのリストを取得するようになりました。 アプリは次のようになります：

![Screenshot](screenshot.png)
