# 試してみよう

これまでのことを試してみましょう。 次のようにしてアプリケーションを実行します。

```bash
nf start
```

ブラウザで `http://localhost:3000`に行きます。 ユーザーの一覧が表示されます。

![screenshot](09-list.png)

`http://localhost:3000/#players/2`に行くと、1人のユーザーが表示されます。

![screenshot](09-edit.png)
